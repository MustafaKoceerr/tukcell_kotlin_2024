# Todo Projesi

Bu proje, SQLite veritabanı kullanılarak geliştirilmiş bir basit Todo uygulamasıdır. Kullanıcılar, görev ekleyebilir, görevleri güncelleyebilir ve silebilir.

## Özellikler

- Görev ekleme
- Görev güncelleme
- Görev silme
- Tüm görevleri listeleme
